# 会话初始化 — 每次对话开始时执行

按顺序完成初始化后，输出确认信息，再等待用户指令。

## 执行步骤

1. 读取 `.ai/SKILL.md` → 锁定全局强制规则（阿里规范强制项）
2. 读取 `.ai/README.md` → 了解技术栈、分层结构、统一响应格式、错误码体系
3. 根据用户需求，定位目标包，读取包级 `SKILL.md`（层职责约束）
4. 在包 `index.json` 中定位目标类，读取类目录的 `index.json`（方法快照）
5. 读取类级 `SKILL.md`（类专属规则）

## 确认输出格式

✅ 全局约束已加载（阿里规范嵩山版）
✅ 目标层：[层名] / 目标类：[类名]
✅ 已有方法：[方法名列表]
✅ 本次任务：[任务描述]
⚠️  注意事项：[本次特别需要关注的规则，如涉及事务/并发/金额等]

完成后等待用户指令，禁止提前生成代码。
